# 全国招标信息获取方案

## 方案概述

由于官方平台API接口不对个人/企业开放，本系统提供以下三种免费获取数据的方式：

### 方式一：数据导入（推荐）

使用八爪鱼、后羿采集器等工具采集数据后，通过本系统的导入接口入库。

#### 步骤

1. **下载采集工具**
   - 八爪鱼采集器（免费版）：https://www.bazhuayu.com
   - 后羿采集器（免费版）：https://www.houyicaiji.com

2. **采集数据**
   - 目标网站：全国公共资源交易平台、中国政府采购网、各省公共资源交易平台
   - 导出格式：JSON

3. **导入数据**
   ```bash
   cd server
   npx tsx src/scripts/import-data.ts --file=你的数据文件.json
   ```

4. **查看数据模板**
   ```bash
   npx tsx src/scripts/import-data.ts --template
   ```

#### 支持的字段映射

| 源字段 | 目标字段 | 说明 |
|--------|----------|------|
| 标题/公告标题/项目名称 | title | 公告标题 |
| 项目编号/招标编号 | project_code | 项目编号 |
| 预算金额/采购预算 | budget | 预算金额 |
| 发布时间/发布日期 | publish_date | 发布日期 |
| 采购人/招标人/业主 | purchaser_name | 采购单位 |
| 联系人 | contact_person | 联系人 |
| 联系电话/联系方式 | contact_phone | 联系电话 |
| 省份/地区 | province | 省份 |
| 公告类型 | announcement_type | 公告类型 |

### 方式二：网页爬虫

系统已实现网页爬虫模块，但由于政府网站反爬虫保护，需要根据实际情况调整。

#### 爬虫文件结构

```
server/src/crawler/
├── types.ts        # 类型定义
├── config.ts       # 配置（省份、类型映射）
├── base.ts         # 爬虫基类
├── ggzy.ts         # 全国公共资源交易平台爬虫
├── ccgp.ts         # 中国政府采购网爬虫
├── storage.ts      # 数据存储
├── scheduler.ts    # 定时任务调度
└── index.ts        # 模块导出
```

#### 使用方式

```bash
# 查看数据统计
npx tsx src/scripts/run-crawler.ts --stats

# 增量爬取
npx tsx src/scripts/run-crawler.ts --pages 10

# 按省份爬取
npx tsx src/scripts/run-crawler.ts --province=220000 --pages 5
```

### 方式三：免费API（有限额度）

部分平台提供免费试用额度：

| 平台 | 免费额度 | 说明 |
|------|----------|------|
| APISpace | 450次/月 | 需注册获取API Key |
| 思通数据 | 联系获取 | 提供政府采购数据API |

## 合规说明

### 法律依据

根据《招标公告和公示信息发布管理办法》（国家发展改革委2017第10号令）：

> 招标公告和公示信息属于**依法必须公开的信息**，任何单位和个人不得非法干涉、限制其发布。

### 合规使用要求

| 允许 ✅ | 禁止 ❌ |
|---------|---------|
| 爬取公开招标公告 | 转售数据牟利 |
| 用于信息展示 | 冒充官方平台 |
| 标注数据来源 | 篡改数据内容 |
| 控制请求频率 | 恶意攻击服务器 |

## 快速开始

### 1. 导入示例数据

```bash
cd server
npx tsx src/scripts/import-data.ts --file=data-template.json
```

### 2. 查看数据统计

```bash
npx tsx src/scripts/run-crawler.ts --stats
```

### 3. 使用八爪鱼采集

1. 下载八爪鱼采集器
2. 创建采集任务，目标网站选择任一公共资源交易平台
3. 配置采集字段（参考上方字段映射表）
4. 导出JSON格式数据
5. 使用导入脚本入库
