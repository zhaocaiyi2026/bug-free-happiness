| 项目 | 内容 |

| 平台名称 | 中国政府采购网 |
| 官方网站 | http://www.ccgp.gov.cn |

4. **等待审核**
   - 审核周期：约1-2周
   - 审核通过后，以邮件方式发送接口调试地址、调试账号

## 二、中国招标投标公共服务平台

### 1. 基本信息

| 项目 | 内容 |
|------|------|
| 平台名称 | 中国招标投标公共服务平台 |
| 官方网站 | http://www.cebpubservice.com |
| 主管部门 | 国家发改委 |
| 数据类型 | 招标项目、招标公告、开标记录、中标候选人、中标结果等17类 |
| 覆盖范围 | 对接31个省级平台+12个部委数据库 |
| 更新频率 | 日均更新超8万条 |
| 接口规范 | 20号令数据交互接口 |
| 费用 | **免费** |

### 2. 申请流程

1. **平台注册**
   - 访问 http://www.cebpubservice.com
   - 点击"对接注册"
   - 填写平台信息（企业信息、应用用途等）

2. **提交审核**
   - 阅读并同意《注册协议》
   - 提交真实准确的信息
   - 等待对接专员审核

3. **获取接口信息**
   - 审核通过后获取平台编码
   - 获取接口文档和调试账号

### 3. 对接要求

- 需要提供前置机服务器资源
- 需要开发推送数据的接口程序
- 需要实时推送招标投标数据
- 需要监测数据问题并及时处理

### 4. 联系方式

- 官网对接说明：http://www.cebpubservice.com/system/explain20/index.shtml
- 对接咨询：通过官网"联系我们"获取

## 五、环境变量配置

在 `.env` 文件中配置以下环境变量：

```bash
# 中国政府采购网
CCGP_API_KEY=your_ccgp_api_key
CCGP_SECRET=your_ccgp_secret

# 中国招标投标公共服务平台
CEBPUB_API_KEY=your_cebpub_api_key

# 全国公共资源交易平台
GGZY_API_KEY=your_ggzy_api_key

# APISpace（备用数据源）
APISPACE_API_KEY=your_apispace_api_key

# 数据同步开关
ENABLE_DATA_SYNC=true
```

## 七、注意事项

1. **合规使用**
   - 数据仅用于合规的招标信息服务
   - 不得转售或泄露数据
   - 遵守各平台使用协议

2. **技术对接**
   - 按照官方接口规范开发
   - 实现签名验证机制
   - 做好错误处理和重试

3. **运维管理**
   - 定期检查接口状态
   - 监控数据同步日志
   - 及时处理异常情况
