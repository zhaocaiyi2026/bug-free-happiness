# 后羿采集器配置 - 吉林省政府采购网

## 网站信息

- **网站名称**: 吉林省政府采购网
- **网站地址**: http://www.ccgp-jilin.gov.cn
- **页面类型**: Vue.js 动态网页
- **关键配置**: 必须开启"动态渲染"

---

## 采集配置

### 基础设置

| 配置项 | 值 |
|--------|-----|
| 动态渲染 | ✅ 开启 |
| 等待时间 | 5000毫秒 (5秒) |
| 采集模式 | 网页采集 |

---

### XPath 配置

#### 列表页配置

**示例URL**: http://www.ccgp-jilin.gov.cn/site/list?parentId=550068

| 字段名 | XPath | 说明 |
|--------|-------|------|
| 标题 | `//a[contains(@class, 'title')]/text()` | 公告标题 |
| 链接 | `//a[contains(@class, 'title')]/@href` | 详情页链接 |
| 发布时间 | `//span[contains(@class, 'date')]/text()` | 发布日期 |
| 公告类型 | `//span[contains(@class, 'type')]/text()` | 公告类型 |

#### 详情页配置

| 字段名 | XPath | 说明 |
|--------|-------|------|
| 标题 | `//h2/text()` | 公告标题 |
| 来源 | `//span[contains(text(), '来源')]/following::text()[1]` | 发布机构 |
| 发布时间 | `//span[contains(text(), '发布时间')]/following::text()[1]` | 发布日期 |
| 正文内容 | `//div[contains(@class, 'content')]//text()` | 全部文本 |
| 正文HTML | `//div[contains(@class, 'content')]` | 含表格图片 |
| 图片链接 | `//div[contains(@class, 'content')]//img/@src` | 所有图片 |
| 表格数据 | `//div[contains(@class, 'content')]//table` | 所有表格 |
| 附件链接 | `//a[contains(@href, '.docx') or contains(@href, '.pdf')]/@href` | 附件下载地址 |
| 附件名称 | `//a[contains(@href, '.docx') or contains(@href, '.pdf')]/text()` | 附件标题 |

#### 关键信息提取

```xpath
# 项目编号
//*[contains(text(), '项目编号')]/following::text()[1]

# 项目名称
//*[contains(text(), '项目名称')]/following::text()[1]

# 预算金额
//*[contains(text(), '预算金额')]/following::text()[1]

# 采购人名称
//*[contains(text(), '名 称')]/following::text()[1]

# 采购人地址
//*[contains(text(), '地 址')]/following::text()[1]

# 采购人电话
//*[contains(text(), '联系方式')]/following::text()[1]

# 采购代理机构
//*[contains(text(), '采购代理机构')]/following::text()[1]

# 项目联系人
//*[contains(text(), '项目联系人')]/following::text()[1]

# 开标时间
//*[contains(text(), '开标时间')]/following::text()[1]

# 投标截止时间
//*[contains(text(), '截止时间')]/following::text()[1]
```

#### 万能XPath（备用）

```xpath
# 获取整个页面所有文本
//body//text()[normalize-space()]

# 获取整个页面HTML
//body

# 获取所有链接
//a/@href

# 获取所有图片
//img/@src
```

## 采集流程

### 步骤1：创建采集任务

```
1. 打开后羿采集器
2. 新建任务 → 输入URL
3. 选择"网页采集"
```

### 步骤2：配置动态渲染（关键！）

```
1. 进入"高级设置"
2. 开启"动态渲染" 或 "JS渲染"
3. 设置等待时间：5000毫秒
```

### 步骤3：添加字段

按照上方XPath配置表添加字段

### 步骤4：测试采集

```
点击"测试采集" → 查看结果 → 调整XPath
```

### 步骤5：导出数据

```
支持格式：JSON、CSV、Excel
推荐格式：JSON（方便导入系统）
```

---

## 导入本系统

采集完成后，导出JSON格式，使用导入脚本入库：

```bash
cd server
npx tsx src/scripts/import-data.ts --file=采集数据.json
```

## 注意事项

1. **必须开启动态渲染**：该网站使用Vue.js，内容是JavaScript动态加载
2. **设置足够等待时间**：建议5秒以上
3. **控制采集频率**：避免被封IP，建议间隔2-3秒
4. **测试XPath**：采集前先用浏览器F12测试XPath是否正确
5. **数据清洗**：采集后可能需要清洗空格、换行符等
