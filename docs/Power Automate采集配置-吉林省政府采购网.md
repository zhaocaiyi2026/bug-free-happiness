## 特殊字段正则表达式

### 提取正文内容
```
正则: 项目概况([^<]*(?:<[^>]+>)*[^<]*)*(?=一、项目基本情况)
说明: 提取项目概况部分
```

### 提取所有段落文本
```
正则: <p[^>]*>([^<]+)</p>
说明: 匹配所有<p>标签内容
```

### 提取附件链接
```
正则: href="([^"]*\.(docx|pdf|xls|xlsx))"
说明: 匹配文档附件链接
```

### 提取图片链接
```
正则: src="([^"]*\.(jpg|png|gif))"
说明: 匹配图片链接
```

## 提取多个字段的技巧

如果需要提取多个同名字段（如多个"联系方式"）：

### 方法1: 使用索引
```
操作: 执行正则表达式
正则: 联系方式[：:]([^<\n]+)
选项: 勾选"查找所有匹配项"
结果: %联系方式列表% (数组)

操作: 获取列表项
列表: %联系方式列表%
索引: 0 (第一个) 或 1 (第二个)
结果: %采购人电话% 或 %代理机构电话%
```

### 方法2: 限定上下文
```
提取采购人电话（在采购人信息块内）:
正则: 采购人信息[\s\S]*?联系方式[：:]([^<\n]+)

提取代理机构电话（在代理机构信息块内）:
正则: 采购代理机构信息[\s\S]*?联系方式[：:]([^<\n]+)
```
