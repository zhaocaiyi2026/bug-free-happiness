# 招标数据合规解决方案

## 问题分析

| 问题 | 说明 |
|------|------|
| 人工录入 | 时间成本高，效率低 |
| 官方 API | 不开放商业用户 |
| 直接爬虫 | 违反 robots.txt，法律风险高 |

## 合规解决方案

### 方案 1：知了标讯 API（推荐）

**官网**: https://www.zhiliaobiaoxun.com

| 项目 | 说明 |
|------|------|
| **费用** | 基础 API 永久免费，按条付费 0.1-0.2 元/条 |
| **数据来源** | 10 万+ 官方网站，已获合法授权 |
| **覆盖范围** | 全国 3 亿+ 条招标数据 |
| **合规性** | ✅ 数据合法授权 |

**API 接口**:
- 标讯查询 API
- 公司信息 API
- 项目详情 API
- 实时推送（延迟 < 10 分钟）

### 方案 2：招标采购导航网

**官网**: https://www.chinabidding.cn

| 项目 | 说明 |
|------|------|
| **费用** | 免费基础版，付费增值服务 |
| **历史数据** | 24 年历史数据库 |
| **日均更新** | 26 万+ 条 |
| **合规性** | ✅ 国家中小企业公共服务示范平台 |

### 方案 3：浏览器插件辅助

**原理**: 用户正常浏览网站时，插件自动提取数据

```
用户浏览公告 → 插件提取 → 发送到 API → 入库
```

**优势**:
- ✅ 用户行为合法
- ✅ 无爬虫风险
- ✅ 效率提升 10 倍

### 方案 4：官方权威平台（免费）

| 平台 | 说明 | 链接 |
|------|------|------|
| 中国招标投标公共服务平台 | 国家级，权威性最高 | http://www.cebpubservice.com |
| 全国公共资源交易平台 | 覆盖 31 省 400+ 子平台 | http://www.ggzy.gov.cn |
| 中国政府采购网 | 财政部指定 | http://www.ccgp.gov.cn |

**注意**: 这些平台数据免费，但无 API，需手动查看。

## 推荐实施路径

1. **短期**: 使用知了标讯 API 获取数据（免费额度）
2. **中期**: 开发浏览器插件辅助采集
3. **长期**: 申请官方数据接口授权

## 法律风险提示

| 行为 | 风险等级 |
|------|----------|
| 使用合规第三方 API | ✅ 无风险 |
| 用户手动复制数据 | ✅ 无风险 |
| 爬取政府采购网站 | ⚠️ 高风险（违反 robots.txt） |
| 商业使用爬取数据 | ❌ 极高风险 |
